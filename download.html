NuGet distribution (recommended)

www.nuget.org/packages/FileHelpers/

Manual install via Console

{% highlight PowerShell %} Install-Package FileHelpers {% endhighlight %}

For Excel Storage

{% highlight PowerShell %} Install-Package FileHelpers.ExcelNPOIStorage {% endhighlight %}

Legacy Support (try to use NPOI version instead)

{% highlight PowerShell %} Install-Package FileHelpers.ExcelStorage {% endhighlight %}

or search for "FileHelpers" in the NuGet Package Manager in Visual Studio

Manual Installation via GitHub

Download at GitHub {{ site.libversion }}

You only need to add a reference in your project to the file: FileHelpers.dll.
Tip: remember to copy also the FileHelpers.xml file to get Intellisense support.
