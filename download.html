Installation via NuGet (recommended)

www.nuget.org/packages/FileHelpers

Manual install via Console

{% highlight PowerShell %} Install-Package FileHelpers {% endhighlight %}

or search for "FielHelpers" in the NuGet Package Manager in Visual Studio

Manual Installation via GitHub

Download at GitHub

You only need to add a reference in your project to the file: FileHelpers.dll.
Tip: remember to leave the FileHelpers.xml file to get Visual Studio Intellisense support.

The package contains:

  • The library for all versions of .net
  • More Examples
  • The Wizard
  • Offline Help chm file

Pre-release versions of the library

{% highlight PowerShell %} Install-Package FileHelpers -Pre {% endhighlight %} You can also access the dialy builds at TeamCity in CodeBetter
