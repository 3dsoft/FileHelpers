Attributes: FixedLengthRecord FixedMode.AllowLessChars
Options when working with fixed files where not all records have same length:
Input.txt
{% highlight csharp %} 01010 Alfreds Futterkiste 13122005 12399 Ana Trujillo Emparedados y 23012000 00011 Antonio Moreno Taquería 042001 51677 Around the Horn 13051998 99999 Berglunds snabbköp 111999 {% endhighlight %}
//-> The Record Mapping class:
RecordClass.cs
{% highlight csharp %} [FixedLengthRecord(FixedMode.AllowLessChars)] public class Customer { [FieldFixedLength(5)] public int CustId; [FieldFixedLength(30)] [FieldTrim(TrimMode.Both)] public string Name; [FieldFixedLength(8)] [FieldConverter(ConverterKind.DateMultiFormat, "ddMMyyyy", "MMyyyy")] public DateTime AddedDate; } {% endhighlight %}
//-> Iterating over the records:
Example.cs
{% highlight csharp %} var engine = new FixedFileEngine(); Customer[] result = engine.ReadFile("input.txt"); foreach (var detail in result) { Console.WriteLine(" Client: {0}, Date: {1}", detail.CustId, detail.AddedDate.ToString("dd-MM-yyyy")); } {% endhighlight %}
