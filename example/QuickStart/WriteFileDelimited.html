QuickStart: Write Delimited File

Example of how to write a Delimited File

To write an output file separated by a |:

Output.txt
{% highlight csharp %} 1|AIRG|01052009|82.43 2|JSYV|02052009|12.22 {% endhighlight %}
RecordClass.cs
{% highlight csharp %} /// /// Layout for a file delimited by | /// [DelimitedRecord("|")] public class Orders { public int OrderID; public string CustomerID; [FieldConverter(ConverterKind.Date, "ddMMyyyy")] public DateTime OrderDate; public decimal Freight; } {% endhighlight %}

Finally you must to instantiate a FileHelperEngine and write the file

Example.cs
{% highlight csharp %} var engine = new FileHelperEngine(); var orders = new List(); orders.Add(new Orders() { OrderID = 1, CustomerID = "AIRG", Freight = 82.43M, OrderDate = new DateTime(2009, 05, 01) }); orders.Add(new Orders() { OrderID = 2, CustomerID = "JSYV", Freight = 12.22M, OrderDate = new DateTime(2009, 05, 02) }); engine.WriteFile("Output.Txt", orders); {% endhighlight %}

The classes you use could come from anywhere, Linq to Entities, SQL database reads, or in this case classes created within an application
